Hoxica Nexus Realm Server 紹介

Hoxica Nexus Realm Serverはサバイバルサーバー・クリエイティブサーバー・ミニゲームサーバーという様々なサーバーが用意されているマルチサーバーです。友達といつでも気軽に遊べ、初心者の人でもすぐに楽しめる環境も目指しております。

Hoxica Nexus Realm Server 参加方法

お使いのエディションに合わせて、以下の手順で参加してください。

☕

◉ Java版

  1. 「マルチプレイ」を選択
  2. 「サーバーを追加」を押す
  3. サーバー名は任意、アドレス欄に「mc.cg.gy」
  4. 追加されたサーバーに接続
📱

◉ 統合版（PS5,Xbox,Switch,スマホ等）

  1. 「プレイ」を選択
  2. 「フレンド」を選択
  3. 「silicaguild」 にフレンド申請
  4. 数秒待ってから参加
